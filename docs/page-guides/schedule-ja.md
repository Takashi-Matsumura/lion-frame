## カレンダーの使い方

カレンダーは、個人の予定管理を中心に、祝日・会社イベントをまとめて確認できる画面です。

### 画面構成

画面は上下2つのエリアで構成されています。

- **上段: カレンダーグリッド** — 月間カレンダー。予定がある日にはカテゴリ色のドットが表示されます
- **下段: 日付詳細パネル** — 選択した日の予定一覧とタイムラインを表示します

![カレンダー全体画面](schedule-overview.png)

> 画面が狭い場合は、ヘッダー右端の折りたたみボタン（⇕アイコン）でカレンダーグリッドを折りたたむと、下段の詳細パネルを広く使えます。

---

### カレンダーナビゲーション

ヘッダーには以下の操作ボタンがあります。

| ボタン | 機能 |
|--------|------|
| **◀** / **▶** | 前月 / 翌月へ移動 |
| **今日** | 今日の日付に戻る |
| **1** / **2** | 1ヶ月表示 / 2ヶ月表示の切り替え |
| **⇕** | カレンダーグリッドの折りたたみ / 展開 |

![ヘッダーナビゲーション](schedule-header.png)

---

### 1ヶ月表示と2ヶ月表示

**1ヶ月表示**では、選択した月のカレンダーが大きく表示されます。

**2ヶ月表示**（デフォルト）では、当月と翌月が1つのグリッドに重ねて表示されます。各セルは対角線で分割され、**左上が当月**、**右下が翌月**の日付です。

![2ヶ月表示](schedule-dual-view.png)

---

### 日付の色分け

カレンダー上の日付は、以下のルールで色分けされます。

- **赤色**: 日曜日・祝日
- **青色**: 土曜日
- **黒丸（白抜き数字）**: 今日の日付
- **背景ハイライト**: 選択中の日付

祝日がある場合は、日付の下に祝日名が小さく表示されます。

---

### 予定のカテゴリと色

予定にはカテゴリを設定でき、カレンダー上ではカテゴリごとの色のドットで表示されます。

| カテゴリ | 色 |
|---------|-----|
| 休暇 | 黄色 |
| 仕事 | オレンジ |
| 会議 | 青 |
| 来客 | 紫 |
| 出張 | ピンク |
| その他 | グレー |

会社イベントは別の色体系で表示されます。

| 種別 | 色 |
|------|-----|
| イベント | ティール |
| 締め切り | 赤 |
| 期間 | エメラルド |

![カテゴリ色の表示例](schedule-category-colors.png)

---

### 日付詳細パネル

カレンダーで日付をクリックすると、下段に詳細パネルが表示されます。パネルは左右に分かれています。

**左側: 予定リスト**
- 祝日（赤背景）
- 会社イベント（ティール背景）
- 個人の予定（時間・タイトル・カテゴリ表示）

予定をクリックすると編集ダイアログが開きます。

**右側: タイムライン**
- 7:00〜21:00の時間軸
- 予定がカテゴリ色のブロックで表示される
- 今日の場合は、現在時刻が赤い線で表示される

![日付詳細パネル](schedule-day-detail.png)

---

### 予定の追加

予定を追加するには2つの方法があります。

**方法1: 「予定を追加」ボタン**

日付詳細パネルのヘッダーにある「+ 予定を追加」ボタンをクリックします。選択中の日付の9:00〜10:00が初期値として設定されます。

**方法2: タイムラインでドラッグ**

タイムライン上で時間帯をドラッグすると、その時間帯で予定作成ダイアログが開きます。時間は15分単位でスナップされます。

![タイムラインでドラッグ作成](schedule-drag-create.png)

> 過去の時間帯には予定を作成できません。過去の日付では「予定を追加」ボタンも表示されません。

---

### 予定フォーム

予定の追加・編集では、以下の項目を入力します。

| 項目 | 説明 |
|------|------|
| タイトル | 予定名（必須） |
| 説明 | 詳細メモ（任意） |
| 場所 | 場所の情報（任意） |
| 終日 | ONにすると時間指定なしの終日予定になる |
| 開始 / 終了 | 日時指定（終日OFFの場合） |
| カテゴリ | 休暇・仕事・会議・来客・出張・その他 |

![予定フォーム](schedule-event-form.png)

編集時には「削除」ボタンが表示され、確認ダイアログの後に削除できます。

---

### AIコンシェルジュ

画面の右端にある「AIコンシェルジュ」タブをクリックすると、AIチャットパネルが開きます。

表示中の月の予定データをもとに、AIがスケジュールに関する質問に回答します。

**クイックプロンプトの例:**
- 「今週の予定は？」
- 「今週の来客は？」
- 「スケジュールに不備はない？」

自由にテキストを入力して質問することもできます。会話はクリアボタン（ゴミ箱アイコン）でリセットできます。

![AIコンシェルジュ](schedule-concierge.png)

---

### ヒント

- 2ヶ月表示では翌月の予定も一目で把握できます。先の予定を確認したいときに便利です
- タイムラインのドラッグ操作は、マウスを押したまま上下に動かすだけで予定の時間帯を直感的に指定できます
- 画面が狭いときはカレンダーを折りたたんで、日付詳細パネルのタイムラインを広く表示しましょう
- AIコンシェルジュに「スケジュールに不備はない？」と聞くと、ダブルブッキングなどの問題を検出してくれます

## Cowork スクリーンショット撮影指示

以下の手順で各スクリーンショットを撮影してください。
すべてのスクリーンショットは `public/uploads/page-guides/` に保存してください。

### 事前準備

1. http://localhost:3000 にログイン（admin@lionframe.local / 任意のパスワード）
2. サイドバーから「カレンダー」メニューをクリック
3. ブラウザウィンドウを横幅1280px以上に設定

---

### SS-1: schedule-overview.png
**カレンダー全体画面**

1. カレンダーページが表示された状態で、2ヶ月表示になっていることを確認
2. 今日の日付をクリックして選択状態にする
3. カレンダーグリッドが展開されている状態で、画面全体のスクリーンショットを撮影
4. ファイル名: `schedule-overview.png`

---

### SS-2: schedule-header.png
**ヘッダーナビゲーション**

1. カレンダーページのヘッダー部分（ナビゲーションボタン一列）にフォーカス
2. 「◀」「▶」「今日」「1/2切り替え」「⇕折りたたみ」が全て見える範囲でスクリーンショットを撮影
3. ファイル名: `schedule-header.png`

---

### SS-3: schedule-dual-view.png
**2ヶ月表示**

1. 表示モードが「2」（2ヶ月表示）になっていることを確認
2. カレンダーグリッド部分にフォーカス
3. 当月と翌月の日付が対角線セルで表示されている状態をスクリーンショット
4. ファイル名: `schedule-dual-view.png`

---

### SS-4: schedule-category-colors.png
**カテゴリ色の表示例**

1. 予定が複数登録されている日付を選択（なければ、事前にいくつかの異なるカテゴリで予定を追加しておく）
   - 例: 「会議: チーム定例」（会議/青）、「出張: 大阪出張」（出張/ピンク）、「休暇: 有給」（休暇/黄色）
2. カレンダーグリッド上で、複数色のドットが表示されている日付付近をスクリーンショット
3. 同時に日付詳細パネルの左側リストで各カテゴリの色ドットが確認できるとなお良い
4. ファイル名: `schedule-category-colors.png`

---

### SS-5: schedule-day-detail.png
**日付詳細パネル**

1. 予定がいくつか登録されている日付を選択
2. 下段の詳細パネルが表示された状態
3. 左側（予定リスト）と右側（タイムライン）が両方見える範囲でスクリーンショット
4. タイムライン上にイベントブロックが表示されていると良い
5. 今日の日付なら、赤い現在時刻線も映る
6. ファイル名: `schedule-day-detail.png`

---

### SS-6: schedule-drag-create.png
**タイムラインでドラッグ作成**

1. 未来の日付を選択
2. タイムライン上で、例えば14:00〜15:00あたりをマウスでドラッグ（押したまま下に動かす）
3. 青い選択範囲と時間ラベル（14:00, 15:00）が表示されている瞬間をスクリーンショット
4. ドラッグ中のマウスボタンを離さない状態で撮影する必要がある
5. ファイル名: `schedule-drag-create.png`

> この撮影は難しい場合はスキップしても構いません。

---

### SS-7: schedule-event-form.png
**予定フォーム**

1. 「+ 予定を追加」ボタンをクリック
2. ダイアログが表示されたら、以下を入力:
   - タイトル: 「チーム定例ミーティング」
   - 説明: 「週次の進捗報告」
   - 場所: 「会議室A」
   - カテゴリ: 「会議」を選択
   - 開始: 適当な未来の日時
   - 終了: 開始の1時間後
3. 入力が全て見える状態でダイアログをスクリーンショット
4. ファイル名: `schedule-event-form.png`
5. 撮影後、「キャンセル」で閉じる（保存しない）

---

### SS-8: schedule-concierge.png
**AIコンシェルジュ**

1. 画面右端の「AIコンシェルジュ」縦書きテキストをクリック
2. 右サイドにAIチャットパネルが開く
3. クイックプロンプト「今週の予定は？」をクリック
4. AIの応答が表示されるのを待つ
5. ウェルカムメッセージ + クイックプロンプト、またはAI応答が表示された状態でスクリーンショット
6. ファイル名: `schedule-concierge.png`

---

### 撮影後の作業

1. スクリーンショットを `apps/web/public/uploads/page-guides/` に保存
2. ページガイドのマークダウン内の画像パスを `/uploads/page-guides/schedule-xxx.png` に書き換え
3. PageGuide テーブルに登録（管理画面のページ情報ボタン → 編集モードで貼り付け）
